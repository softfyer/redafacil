# **App Name**: Redação Online

## Core Features:

- Student Registration and Login: Secure registration and login system for students using Firebase Authentication.
- Essay Upload: Students can upload essays in various formats (PDF, JPEG, DOCX, PNG).
- Teacher Login: Teachers access the system using pre-registered credentials.
- Essay Correction Interface: Teachers can download original essays, upload corrected versions, record audio feedback using the Web Audio API, and provide text feedback.
- Payment System Structure: Component to validate payments for essay upload access and Mock payment during development.
- Notifications System: Students receive notifications when their essays are corrected.

## Style Guidelines:

- Primary color: Soft Blue (#4F46E5) to provide a calm and trustworthy feeling.
- Background color: Light grays (#F9FAFB) to give an uncluttered background for educational use.
- Accent color: Educational Purple (#7C3AED) for emphasis and calls to action.
- Body and headline font: 'Inter' (sans-serif) to ensure a clean and modern reading experience.
- Simple and clear icons to represent different actions and statuses.
- Mobile-first, responsive design to ensure accessibility on all devices.
- Subtle transitions and animations to provide feedback and improve user experience.
